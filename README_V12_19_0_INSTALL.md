# Brian English Studio V12.19.0 — Department Command Center

## Nâng cấp
- Thiết kế lại trang Tổ chuyên môn theo bố cục điều hành 16:9 hiện đại.
- Hero tương phản cao, hiển thị vai trò, số việc mở, hồ sơ chờ duyệt và trạng thái lưu trữ.
- Bốn thẻ tổng quan được làm mới, rõ số liệu và dễ thao tác.
- Dashboard kế hoạch, công việc, hồ sơ và báo cáo thích ứng 2–4 cột theo màn hình.
- Thanh module trở thành thanh điều hướng dính, dễ chuyển giữa các chức năng.
- Giữ nguyên dữ liệu, phân quyền TTCM/giáo viên, Supabase và toàn bộ nghiệp vụ cũ.

## Cài đặt
Không cần chạy SQL. Chép bản update-only vào thư mục dự án, chạy `npm ci`, `npm run verify:v12.19.0`, rồi push GitHub.
