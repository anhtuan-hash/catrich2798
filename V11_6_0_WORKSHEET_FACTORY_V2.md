# Brian English Studio V11.6.0

## Worksheet Factory V2 — Structured Learning Pack Workbench

Worksheet Factory V2 replaces the former linear source/config/generate/export screen with a seven-stage production workflow and nine functional cards.

### Workflow

1. Source
2. Source Intelligence
3. Learner & objectives
4. Worksheet Blueprint
5. AI section generation
6. Editor and audit
7. Publish and connected workflow

### Build modes

- Source to Worksheet
- Topic to Worksheet
- Lesson Pack Builder
- Existing Worksheet Refiner
- Item Bank Composer
- Batch Worksheet Generator

### Production features

- DOCX, PDF, PPTX, XLSX, TXT, Markdown, HTML and transfer input
- Source facet approval before AI generation
- 21 activity types
- Sequential section generation with offline fallback
- Section/item editor, version history and teacher approval
- Answer, duplicate, source-alignment, blueprint and print-readiness audit
- A4 preview and eight layout templates
- Student/Teacher DOCX, HTML, JSON and batch ZIP variants
- Teacher Library, Item Bank and Connected Workflow
- BURS Comfortable responsive layout

No new Supabase migration is required.
