## Vietnam Tax Calculator

Vietnam Tax Studio was integrated and adapted from the open-source project **Vietnam Tax Calculator** by **Viet Vu Danh**:

- Source project: `https://github.com/vietvudanh/vietnam-tax-2025`
- Original project documentation states that the project is distributed under the MIT License.

The Brian English Studio integration replaces the original standalone shell and chart dependencies with a dependency-free React module, updates the 2026 calculation assumptions and adds the Brian English design system, bilingual interface, responsive layout, print report and CSV export.
