# Brian English Studio V12.25.0

## Provider Command & Appearance Studio

- Chọn provider mặc định bằng một lần bấm.
- Tìm kiếm và lọc provider theo khuyến nghị, đã có key hoặc miễn phí.
- Lưu cấu hình và đặt mặc định trong cùng một thao tác.
- Appearance Studio có xem trước trực tiếp.
- Bổ sung phong cách bề mặt, độ bo, bóng, nền và kiểu chuyển cảnh.
- Đồng bộ tùy chọn giao diện theo tài khoản như các cài đặt hiện có.
- Không cần SQL.
