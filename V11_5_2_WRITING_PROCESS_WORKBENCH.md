# Brian English Studio V11.5.2 — Writing Studio Process Writing Workbench

Writing Studio was rebuilt from a generic prompt page into a complete process-writing environment for Vietnamese upper-secondary English teaching.

## Workflow

1. Build Mode
2. Writing Task & Genre
3. Learner & Context
4. Success Criteria & Rubric
5. Planning & Language Studio
6. Draft Editor & Quality Review
7. Publish & Connected Workflow

## Nine functional cards

- Build Mode
- Writing Task & Genre
- Learner & Context
- Source & Input
- Success Criteria & Rubric
- Planning & Language Studio
- AI Writing Coach
- Draft Editor & Quality Review
- Publish & Connected Workflow

## AI Writing Coach

The coach uses the active Brian AI provider and model. Tasks include prompt analysis, idea generation, outline creation, language support, model analysis, rubric feedback, selected-text rewriting, differentiation and custom coaching. AI edits are proposed and require teacher or learner approval.

## Data and integration

Projects autosave per Brian user in scoped local storage. Teacher Vault, version history, DOC/HTML export, Transfer Inbox and the `bes-writing-pack/1.0` Connected Workflow schema are included. No new Vercel Function or database migration is required.
