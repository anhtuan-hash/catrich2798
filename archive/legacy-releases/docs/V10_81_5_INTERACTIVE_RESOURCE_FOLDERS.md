# Brian English Studio V10.81.5 — Interactive Resource Folders

## Changes
- Clicking any category card now opens that folder and automatically scrolls to its file list.
- Previous search, grade and school-year filters are reset when a folder is opened, so uploaded files cannot be hidden accidentally.
- The folder content panel appears immediately below the filters, before AI search.
- Admin sees all uploads in the folder, including pending and revision items.
- Teachers see all approved department resources plus their own uploads.
- Each category card now shows a clear “Mở thư mục →” action.
- A folder header shows the file count and approval status summary.
- Uploading from an opened folder preselects that category.

## Database
No additional SQL is required. Keep the V10.81.4 RLS hotfix already installed.
