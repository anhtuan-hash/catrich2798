Để dùng font cá nhân trên mọi trình duyệt / domain:
1. Sao chép file font cá nhân của bạn vào thư mục này.
2. Đổi tên file thành: BrianGesco.ttf
3. Nếu deploy trực tiếp thư mục dist, cũng sao chép file này vào: dist/fonts/BrianGesco.ttf

Ứng dụng đã được cấu hình @font-face cố định đường dẫn /fonts/BrianGesco.ttf?v=6.8 và áp dụng font cho toàn bộ giao diện.
