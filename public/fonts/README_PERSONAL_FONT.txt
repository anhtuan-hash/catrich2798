Font cá nhân của Brian English Studio

Tên font nội bộ được hệ thống nhận diện: 1FTV HF Gesco / 1FTVHFGesco.

Cách 1: Nếu đã cài font vào máy, app sẽ tự gọi local("1FTV HF Gesco").
Cách 2: Nếu muốn dùng trên Vercel/hosting, tự đặt file font của thầy vào public/fonts/personal-font.ttf trước khi deploy.
Cách 3: Trong app, vào Cài đặt > Font cá nhân > Chọn file font của tôi.

Bản zip này không phát tán lại file .ttf.
