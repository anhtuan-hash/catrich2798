# Brian English Studio V12.0.0 — UI Core Foundation

This release is built from the uploaded V11.6.7 full source and remains a complete Vite project.

## Included
- Semantic UI tokens for color, surface, spacing, radius, elevation, and motion.
- Three adapters: Brian Unified, Android Material 3, and Apple iOS/iPadOS.
- Route-level layout contracts.
- Shared UI primitives for buttons, cards, fields, text areas, and chips.
- A design-language selector inside Settings.
- Persistent selection through localStorage and cross-tab storage events.
- Stable horizontal textarea rules within the new component layer.

## Migration boundary
Existing application pages are intentionally preserved. V12.0 establishes the foundation; subsequent releases can replace legacy controls app-by-app without changing business logic.
