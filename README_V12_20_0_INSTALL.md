# Brian English Studio V12.20.0 — Interactive Department Cards

## Nâng cấp
- Bốn thẻ Tổ chuyên môn được thiết kế lại theo dashboard SaaS hiện đại.
- Lịch tháng có chuyển tháng, chọn ngày và lọc loại sự kiện trực tiếp.
- TTCM có thể đánh dấu hoàn thành công việc ngay trên thẻ.
- Hồ sơ hỗ trợ chọn nhiều tệp hoặc kéo thả để ghi nhận nhanh vào danh mục.
- Gợi ý AI mở thẳng AI Copilot với yêu cầu đã điền sẵn.
- Xuất/nhập JSON, xuất lịch ICS và hồ sơ HTML hoạt động ngay trên thẻ.
- Desktop hiển thị 4 cột; tablet 2 cột; mobile 1 cột.

Không cần chạy SQL. Chép bản update-only vào dự án, chạy `npm ci`, `npm run verify:v12.20.0`, rồi push GitHub.
