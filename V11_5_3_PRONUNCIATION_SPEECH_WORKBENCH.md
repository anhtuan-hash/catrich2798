# Brian English Studio V11.5.3 — Pronunciation Coach Speech Practice Workbench

Pronunciation Coach is rebuilt as a seven-step, nine-card speech-practice workflow for upper-secondary English.

## Core capabilities

- Six modes: sound, word, rhythm, connected speech, intonation and diagnostic practice.
- Focus selector for phonemes, stress, rhythm, weak forms, linking and intonation.
- Practice Blueprint with activity types, model rate, replay and recording limits.
- Browser Speech Synthesis model player with American/British accent selection.
- Browser MediaRecorder, live waveform and optional Speech Recognition transcript.
- Evidence-aware intelligibility score based on target/transcript similarity.
- AI Speech Coach for generation, annotation, analysis and remediation.
- Version history, Teacher Vault, assignment records and connected publishing.

The browser transcript is presented as an intelligibility check, not phoneme-level assessment. Detailed phoneme scoring requires a specialized external service.
