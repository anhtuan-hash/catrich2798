# Brian English Studio V12.29.0 — Lesson Pack Connected Workflow

## Nâng cấp chính
- Xóa thanh bên riêng của Lesson Pack.
- Hero hiện đại, đầy đủ trạng thái Cloud, lớp, thời lượng và số hoạt động.
- Workflow 6 bước có thể nhấn trực tiếp.
- Mỗi bước có không gian làm việc riêng: mục tiêu, nguồn, khung, hoạt động, giao bài và xuất bản.
- Kéo thả chuỗi hoạt động, biên tập từng hoạt động và cân chỉnh thời lượng.
- Kết nối Classroom Delivery, Live Presentation, JSON, HTML và PDF.
- Không cần SQL mới.
