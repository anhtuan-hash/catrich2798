# V10.29 — Speaking Studio, WordGraph Studio, TextCare redesigned

## Updated apps
- Speaking Studio
- WordGraph Studio
- TextCare

## What was redesigned
### 1) New hero sections for all 3 apps
Each app now has:
- a **raised-flat hero illustration**,
- a clearer **3-zone hero layout**,
- a distinct **color identity**,
- compact **status cards** on the right,
- larger, cleaner typography.

### 2) Interactive workflow cards
Added 3 interactive workflow cards under the hero in each app.
These cards now:
- visually highlight the active workflow,
- scroll users to the relevant section,
- make the page easier to scan and use.

### 3) Balanced layout improvements
Adjusted layout so cards no longer feel oversized or stretched to both page edges.
Changes include:
- tighter spacing,
- more balanced max width,
- improved card proportions,
- clearer two-column work areas,
- better 16:9 readability.
