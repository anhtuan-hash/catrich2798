# V10.76 · Conduct lock password and Grade 12 summer averages

## Weekly finalization security

- Manual weekly finalization and reopening now use an in-app password dialog.
- Default password: `PEK@2026`.
- The default password is shown in the Conduct settings so the homeroom teacher knows the initial credential.
- Homeroom teachers can change the password after entering the current password.
- Administrators can reset the class conduct password to the default.
- Custom passwords are stored as SHA-256 hashes, not plain text.
- Locking and reopening require a reason and remain in the conduct audit trail.
- Automatic scheduled finalization remains a trusted system action and does not stop for interactive password input.

## Grade 12 summer weeks

For Grade 12 classes only, the four summer conduct weeks are included in the average:

1. 15/06/2026–20/06/2026
2. 22/06/2026–27/06/2026
3. 29/06/2026–04/07/2026
4. 06/07/2026–11/07/2026

They are included in Midterm I, Semester I and full-year conduct averages. Other grades continue to use the 42 official semester weeks only. Orientation Week 0 remains excluded from averages.
