# Brian TextLab Activities

Web tĩnh tạo hoạt động tương tác từ văn bản.

## Cách chạy
Mở `index.html` trực tiếp trong trình duyệt hoặc kéo cả thư mục lên Netlify/Vercel/GitHub Pages.

## Ghi chú font cá nhân
Nếu muốn dùng font riêng của bạn, đặt file font tại:
`assets/brian-font.ttf`

Vì lý do bản quyền/phân phối font, gói này không đính kèm file font.
