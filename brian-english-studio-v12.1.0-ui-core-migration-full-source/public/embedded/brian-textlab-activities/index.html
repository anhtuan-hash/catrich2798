Hướng dẫn sử dụng Brian TextLab Activities

    
    
      
  1. Chọn một mẫu hoạt động ở mục Choose Template. Có thể tìm nhanh bằng ô search.
    2. 
      
  2. Dán nội dung vào Your Content. Mỗi mẫu có định dạng riêng, xem phần gợi ý ngay dưới template.
    3. 
      
  3. Preview sẽ tự động tạo. Dùng nút + / − để phóng to hoặc thu nhỏ.
    4. 
      
  4. Bấm Looks Good! Finish & Use để khóa hoạt động vào khu vực dùng chính thức.
    5. 
      
  5. Bấm Download my work để tải hoạt động thành file ZIP gồm HTML/CSS/JS, có thể dùng offline hoặc đưa lên Netlify/Vercel.
    6. 
    

  

  
    
    
TextLab Instructions

    
    
      
  1. Choose an activity template. Use search to find a specific activity.
    2. 
      
  2. Paste your text using the suggested input format.
    3. 
      
  3. The preview updates automatically. Use zoom controls if needed.
    4. 
      
  4. Click Looks Good! Finish & Use to finalize the activity.
    5. 
      
  5. Click Download my work to export a standalone ZIP package.
    6. 
    

  

  
    
    
Đưa sản phẩm lên online

    
    
      
  1. Bấm Download my work để tải ZIP hoạt động.
    2. 
      
  2. Giải nén ZIP, bạn sẽ có index.html, style.css, activity.js.
    3. 
      
  3. Kéo cả thư mục lên Netlify Drop, Vercel, GitHub Pages hoặc bất kỳ hosting tĩnh nào.
    4. 
      
  4. Chia sẻ link cho học sinh. Hoạt động không cần database.
    5.
