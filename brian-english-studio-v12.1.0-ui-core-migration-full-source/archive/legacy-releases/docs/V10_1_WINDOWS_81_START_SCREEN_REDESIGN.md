# V10.1 Windows 8.1 Start Screen Redesign

This release redesigns the homepage as a full Windows 8.1-inspired Start Screen.

## What changed

- Home is no longer a landing page; it is a full-screen Start Screen.
- Home hides the website topbar/footer and uses an in-page Start header.
- Apps Hub uses the same tile system and horizontal group navigation.
- Tiles support normal, wide and large sizes.
- Tile groups follow teacher workflow: Brian English, Plan, Create, Assess, Manage.
- Home and Apps use flat blue background, high-contrast tile colors and no shadow/blur/gradient.
- Tile Launch Motion is kept as the app-opening experience.
- Mobile switches to a vertical tile stack for usability.

## Version

10.1.0
