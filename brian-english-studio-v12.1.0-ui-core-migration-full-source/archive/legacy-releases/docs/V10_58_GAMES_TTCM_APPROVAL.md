# V10.58 Games Studio: TTCM approval workflow

- Removed the Games sidebar.
- Removed the game library, quick access, recent sessions, and teacher tip dashboard blocks.
- Retained the hero, game selection board, Load in Studio action, and Live Game preview.
- Custom games can be saved privately to the submitting teacher account.
- Teachers can submit a custom game to TTCM for approval.
- Pending or rejected games are visible only to the submitting teacher and TTCM/Admin.
- Approved games are automatically visible to all approved teachers.
- TTCM/Admin can approve, reject, reconsider, or remove custom games.
- Run `supabase/custom_games_approval_v10_58.sql` once in Supabase SQL Editor.
